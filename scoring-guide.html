🤖 Smart Filters IA v2 ADAPTATIF

L'IA analyse les calls passés (winners vs losers) et génère des règles dynamiques qui évoluent avec le temps.

📏 Règles simples

L'IA teste chaque feature individuellement et trouve les seuils où le taux de loss est significatif.

Features analysées : volMcapH1, sellBuyRatio, c5m, c1h, top10pct, buyRatio, patternScore, c6h, liquidité

🔗 Combos toxiques

L'IA détecte les combinaisons de 2 features qui, ensemble, prédisent une perte. 22 paires testées.

Ex : c5m élevé + c1h élevé, sellBuyRatio élevé + volMcap élevé
